The frontend is built with:

- React 19 — core UI framework (react, react-dom, react-router-dom)
- Tailwind CSS v3 — utility-first styling (tailwindcss, tailwind-merge, tailwindcss-animate)
- Radix UI — headless accessible component primitives (accordion, dialog, dropdown, tabs, etc.)
- shadcn/ui pattern — the combination of Radix UI + class-variance-authority + clsx is the standard shadcn/ui setup
- Recharts — charting/data visualization
- React Hook Form + Zod — form handling and validation
- CRACO — CRA config overrides (build tooling on top of Create React App)
- So in short: React + Tailwind CSS + Radix UI (shadcn/ui components)
